Resources Used to Build My Personal Website

Below is a comprehensive list of all resources, tools, tutorials, and libraries used to create my personal profile website.

1. Development Tools & Platforms

  • Code Editor & Version Control: GitHub - Used for writing, editing, and hosting the code with version control.
  • Quick Code Preview: CodePen - Used for quickly previewing and testing HTML, CSS, and JavaScript as I code.

2. Libraries and Frameworks

  • Google Fonts: Google Fonts - Used for custom fonts to enhance the typography of the site. Specifically, the Poppins font was used for a clean and modern look. View Poppins on Google Fonts

3. Tutorials and Guides

  • HTML Basics: W3Schools HTML Tutorial - Provided a solid foundation in HTML structure and elements.
  • CSS Styling: W3Schools CSS Tutorial - Helped with understanding CSS styles and layout techniques.
  • JavaScript HTML DOM: W3Schools JavaScript HTML DOM - Basic guide to understanding how JS HTML DOM can be used for interactivity.
  • Vue.js Basics: W3Schools Vue.js Tutorial - Used for learning Vue.js and implementing reactive, component-based architecture in the project.
  • Web Development Tutorial: YouTube Tutorial - Step-by-step tutorial on creating simple personal website with HTML and CSS.

4. Design Inspiration and References

  • Design Inspiration: Pinterest - Main design inspiration for my website.
